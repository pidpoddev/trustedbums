# Trusted Bums Search And Citation Coverage

_Last updated: 2026-06-21 by Codex._

## Purpose

This packet defines the legitimate search and citation work for `TB-0109`. The goal is to make `trustedbums.com` easier to discover and verify without buying links, exchanging links, mass-submitting directories, or creating low-quality guest posts.

## Current Proof

- `https://trustedbums.com` is the canonical public host for search proof.
- `https://trustedbums.com/sitemap.xml`, `https://trustedbums.com/robots.txt`, and the IndexNow key file are live on the deployed site.
- Google Search guidance supports root-hosted absolute URL sitemaps and Search Console submission.
- Bing supports IndexNow and Bing Webmaster submission as discovery signals.
- The repo rules ban paid backlinks, reciprocal link exchanges, mass directory submission, thin guest posts, and synthetic citation networks.

## Copy Boundaries

- Describe Trusted Bums as a selective B2B warm-introduction marketplace for hard-to-reach target accounts.
- Link buyers to the homepage strategy-review path and Bums to `/bums/`.
- Do not claim guaranteed meetings, guaranteed revenue, customer-logo proof, payout amounts, or verified customer outcomes unless Legal and the named party approve the exact wording.
- Do not use generic lead-generation, passive-income, affiliate, or scraped-list framing.

## Closure Evidence Required

`TB-0109` can close only when at least three approved external citations are published or owner-console proof confirms equivalent search submission coverage. Each proof entry should include:

- published URL or owner-console run ID;
- target page on `trustedbums.com`;
- approval owner;
- date verified;
- confirmation that no paid, exchanged, mass-directory, thin-guest-post, or synthetic link tactic was used.
